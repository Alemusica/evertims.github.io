Install EVERTims


This section guides you through the installation of EVERT (raytracing library) and EVERTims (raytracing program based on EVERT).

Install EVERTims: EVERT


svn co https://evert.svn.sourceforge.net/svnroot/evert/branches/dev-spat evert

Install EVERTims: EVERTims


svn co https://evertims.svn.sourceforge.net/svnroot/evertims/branches/dev-spat evertims
