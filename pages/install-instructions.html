Install EVERTims


This section guides you through the installation of EVERT (raytracing library) and EVERTims (raytracing program based on EVERT). The symbol // used in this section refers to the root of the EVERTims project.


Install EVERTims on OSX

With a terminal window, go to the root of you EVERTims project and type:

{% highlight bash %} # create source directory mkdir source cd source # get EVERT svn co https://evert.svn.sourceforge.net/svnroot/evert/branches/dev-spat evert # compile EVERT (Xcode) cd evert/build/macos xcodebuild cd ../../.. # get EVERTims svn co https://evertims.svn.sourceforge.net/svnroot/evertims/branches/dev-spat evertims # compile EVERTims (Xcode) cd evertims/build/macos xcodebuild cd ../../.. # back to EVERTims root directory cd .. {% endhighlight %}

Install Blender Addon


This section guides you through the installation of the Blender addon designed for the creation and visual exploration of EVERTims scene.The symbol // used in this section refers to the root of the EVERTims project.

Start by downloading the EVERTims Blender addon from github or go to the root of you EVERTims project and type:

{% highlight bash %} # get Blender addon git clone https://github.com/EVERTims/game_engine_evertims.git {% endhighlight %}
