Install EVERTims raytracing client


The raytracing client is a C++ standalone: as for today, installing will mean downloading and compiling the sources.


Raytracing client install on OSX

Manual installation:
Download evert and evertims source code. Open and compile evert Xcode project (i.e. build/macos/evert.xcodeproj). Same for evertims, making sure both evert and evertims folders are located in the same directory (that's where evertims.xcodeproj expects to find evert).

Automatic installation:
With a terminal window, go to were you want EVERTims installed and type:

{% highlight bash %} # create raytracing client directory mkdir raytracing && cd raytracing # get EVERT curl -sL https://github.com/EVERTims/evert/archive/master.zip | tar xz mv evert-master evert # compile EVERT (Xcode) cd evert/build/macos xcodebuild cd ../../.. # get EVERTims curl -sL https://github.com/EVERTims/evertims/archive/master.zip | tar xz mv evertims-master evertims # compile EVERTims (Xcode) cd evertims/build/macos xcodebuild cd ../../.. # back to EVERTims root directory cd .. {% endhighlight %}

Check installation: same place, type:

{% highlight bash %} ./raytracing/evertims/bin/ims {% endhighlight %}

which should result in the following output:

{% highlight bash %} No such material file: materials.dat Reader object started. socket : OK bind : OK {% endhighlight %}


Raytracing client install on Linux

Manual installation:
Download evert and evertims source code. Compile first evert then evertims.

Automatic installation:
With a terminal window, go to were you want EVERTims installed and type:

{% highlight bash %} # create raytracing client directory mkdir raytracing && cd raytracing # get EVERT curl -sL https://github.com/EVERTims/evert/archive/master.zip | tar xz mv evert-master evert # compile EVERT (Xcode) cd evert/build cmake .. && make cd ../.. # get EVERTims curl -sL https://github.com/EVERTims/evertims/archive/master.zip | tar xz mv evertims-master evertims # compile EVERTims (Xcode) cd evertims/build cmake .. && make cd ../.. # back to EVERTims root directory cd .. {% endhighlight %}

Check installation: same place, type:

{% highlight bash %} ./raytracing/evertims/bin/ims {% endhighlight %}

which should result in the following output:

{% highlight bash %} No such material file: materials.dat Reader object started. socket : OK bind : OK {% endhighlight %}

Install Blender Add-on


Download and install Blender if need be, then download the EVERTims Add-on. Install the add-on from Blender preferences (see e.g. the Blender manual instructions or one of the methods presented here for details). You should end-up with the add-on nested in the Blender 3D view > Toolbox (press T while hovering the mouse over the 3D view if the toolbox is not displayed):

To complete the installation, setup the paths to the “Embedded auralization client” and the “Embedded raytracing client” in the add-on panel.
