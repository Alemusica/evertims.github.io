About the EVERTims project

EVERTims is an open source framework for real-time auralization. Given a virtual room (geometry and materials), the main program simulates how acoustic waves propagates there, from defined sources to listeners objects. The results of this simulation feed a real-time auralization engine. In the end, the framework enables virtual environment exploration, completing traditional visual reandering with realistic acoustics propagation.

EVERTims framework is composed of three components: The EVERTims program itself (simulate audio propagation), a Blender addon (design, visualize, and explore virtual environment), and a Pd addon (render real-time auralization based on EVERTims simulated propagation)
