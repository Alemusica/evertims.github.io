About the EVERTims project


EVERTims is an open source framework for real-time auralization. Given a virtual room (geometry and materials), the main EVERTims program simulates how acoustic waves propagates there, from defined sources to listeners objects. The results of this simulation can be used to feed a real-time auralization engine. With the Blender Game Engine being used to create and explore the virtual room, and a PureData dataflow handling the auralization, the EVERTims framework enables virtual environment exploration, completing traditional visual rendering with realistic acoustics propagation.

EVERTims framework is composed of three components: The EVERTims program itself (simulate audio propagation), a Blender module (design, visualize, and explore virtual environment), and a Pd patch (render real-time auralization based on EVERTims simulated propagation). More on these in the Getting Started section.
